# Jack Russell Tic Tac Toe

A two-player Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript featuring a neumorphic soft-UI design.

## Features

- **Neumorphic design** — soft extruded and inset shadows throughout for a modern, tactile feel
- **3D board tilt** — the board tilts left for Player X and right for Player O using CSS 3D transforms
- **Player glow** — a red glow indicates X's turn, blue indicates O's turn
- **Score tracking** — wins for X, O, and draws are tracked across rounds
- **Winning highlight** — the winning combination is outlined when a player wins

## How to Play

Open `index.html` in any modern browser. No build step or dependencies required.

Two players take turns clicking cells to place their mark. First to get three in a row — horizontally, vertically, or diagonally — wins. Click **New Game** to reset the board.
